# Vibe Check

## What is a Vibe Check?

Not a code review.
Not a security audit.
A *vibe check*.

How does this code *feel*?

### Energy Assessment
- **Chaotic energy** - Is this code held together by hope and prayers?
- **Chill vibes** - Is this code relaxed, readable, zen?
- **Anxious energy** - Does this code have too many edge case handlers?
- **Confident energy** - Does this code know what it's doing?

### Aesthetic Score
- Does the code *look* good?
- Is there visual rhythm?
- Are the names pleasant to say out loud?
- Would you put this on your portfolio?

### Emotional Residue
- Can you feel the developer's frustration?
- Are there passive-aggressive comments?
- Does it have "wrote this at 3am" energy?
- Can you sense deadlines in the code?

### Social Compatibility
- Would you pair program with this code?
- Would you introduce this code to your friends?
- Is this code embarrassing at parties?
